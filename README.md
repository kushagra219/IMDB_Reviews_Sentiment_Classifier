# IMDB_Reviews_Sentiment_Classifier

### Dataset - https://www.kaggle.com/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews 

### Concepts Learnt
* tensorflow Tokenizer
* embedding layer
